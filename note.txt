---------------------1

Модель способна предсказать только некоторые мРНК по их комплиментарности и обучалась на других растениях.
Некоторые мРНК, которые мы отсевенировали, она предсказать не способна (потому что может их ещё нет в базе),
а значит у неё есть сет возможных мРНК, которые она может отдать как ответ.
Это даёт нам следующую ситуацию: если в списке модели нет мРНК, которую предсказала коррелация, это не значит, что предсказание корреляции плохое.
Сравнить 819 строк из модели с 267 804 строками связей miRNA - mRNA как-то странно, значит есть следующие варианты:

1) отобрать топ кластеров по корреляции - можно, не сложно
2) отобрать внутри кластеров топ mRNA по корреляции - сложно, но придётся всё равно потом для оборащения в 3ей части
3) отобрать внутри кластеров только те mRNA, которые в принципе предсказывала модель
4) внутри каждого кластера выбрать топ 20 по mad - нормально, сделаю

что сделала в итоге: 
сначала пришлось отфильтровать всё из статистически значимой даунрегуляции так, чтобы остались mRNA, которые были предсказаны моделью.
потом для каждого кластера были выделены топ n% по характеристике mad, чтобы увеличить вероятность не случайности совпадений, если таковые будут.
дальше были сделаны для 22% и для 100% первых по mad барплоты совпадений предсказаний для miRNA.
Ещё для 100% был сделан барплот для miRNA по jaccard index.

результат: ничего нам это не дало)
